📝 To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript. This project helps users organize their daily tasks efficiently and keep track of their work.

🚀 Features

➕ Add new tasks easily

❌ Delete tasks when completed

💾 Tasks are saved using Local Storage (data remains even after refreshing the page)

🎨 Clean and responsive user interface

🛠️ Technologies Used

HTML for structure

CSS for styling

JavaScript for functionality (DOM manipulation & Local Storage)
